# Quality Criteria: Twitter Engagement

Este checklist é usado pela **Rita Revisão** para avaliar cada sugestão de comentário.

| Critério               | Meta       | Descrição                                                                 |
|----------------------|------------|-------------------------------------------------------------------------|
| Relevância do Post   | 8/10+      | O post original trata realmente de IA/Automação de forma valiosa?         |
| Tom de Voz           | 7/10+      | O tom é descontraído, inovador e profissional (estilo Gbm Tecnologia)?   |
| Força do Gancho      | 8/10+      | A primeira frase é capaz de parar o scroll (scroll-stop)?                |
| Engajamento (CTA)    | 9/10+      | O comentário termina com uma pergunta ou convite à resposta natural?      |
| Formatação (Dwell)   | 7/10+      | O texto usa quebras de linha para facilitar a leitura no mobile?         |
| Segurança de Marca   | 10/10      | Não há conteúdo ofensivo, polêmico ou que prejudique a imagem da Gbm?     |

## Regras de Veredito
- **APROVAR:** Média >= 8/10 e nenhum critério abaixo de 5/10.
- **REPROVAR:** Média < 8/10 ou qualquer critério abaixo de 5/10.
- **REPROVAR (Crítico):** Falha no critério Segurança de Marca (10/10 obrigatório).
