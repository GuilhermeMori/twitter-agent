# Quality Criteria: Social Metrics Analysis

## Data Integrity
- [ ] Account matches the requested profile URL.
- [ ] Metric counts are non-negative and realistic.
- [ ] All dates are within the requested period.

## Analysis Quality
- [ ] Growth is calculated using: ((Current - Previous) / Previous) * 100.
- [ ] Comparison baseline is explicitly stated.
- [ ] Insights explain *why* a metric moved (if data allows).

## Output Presentation
- [ ] Report is in the user's preferred language (Portuguese).
- [ ] Tables are properly aligned in Markdown and CSV.
- [ ] Executive summary contains at least 3 bullet points.
